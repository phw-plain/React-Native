# React-Native Study

### Create
참고 사이트: https://reactnative.dev/docs/environment-setup
```

  npx create-expo-app AwesomeProject

  cd AwesomeProject
  npm start # you can also use: npx expo start

```


### Start
```

  cd 프로젝트명
  
  npm install
  
  npm start
 
  expo go 앱 실행 후 QR CODE 스캔

```
